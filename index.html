Mobile and Responsive

Web design and development has changed rapidly over the last few years as the use of mobile devices has increased. Rather than simply displaying a sites content in a predetermined width and layout modern websites are expected by the user to give the best user experience for the device being used.

Responsive web design is required to allow the website to change its page width and layout depending on the screen size of the device.
