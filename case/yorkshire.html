Yorkshire

For Yorkshire.com I was tasked with creating the UI for the new website using a number of advanced techniques (for the time) using Jquery, CSS and HTML.

Whist still being accessible to earlier versions of IE such as version 7 and even 6. The main goal was create the UI to be fast and as usable as possible from a challenging design concept. I was able to work with the designer and indicate any potential issues the design posed when it came to creating the HTML.

  • Year: 2009
  • Role(s): UI Designer and Developer
  • Agency / Client: Mccann Erikson

Challenges

Footer

This site brought with it quite a challenging footer design. The idea was that the hands gave the indication that the user was holding a leaflet about Yorkshire and what it has to offer.

The footer had the follow the content at the footer of the page.

Rounded corners for early IE

The site still had to be compatible with early versions of IE but offer corners with curves.

The solution after much testing was to use a Jquery plugin that created curvy corners which worked with IE 6 and 7 and didn't require multiple images.

Design

The overall design was to give the impression to the user that they were holding an information leaflet whilst standing in a location that look very much like a location in Yorkshire.

Technology

Jquery and Jquery UI were used to enhance and add interactive features to the site

The site had to be put on to an existing umbraco CMS.

Outcome

My role in the site development was to create a front end design to be placed into the existing umbraco CMS by a .net developer. The result was pleasing after much research and some trial and error to overcome some issues. The site went on to win multiple awards for design and technical achievement.
