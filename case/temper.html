Temper Trap

The site was intended to be an information source for fans to catch up on latest news and events from the band. It was decided the best way to get this functionality quickly and within a restricted time scale was to use Drupal CMS.

As Drupal is very flexible it suited the quite challenging functionality concepts whilst being fairly straight forward to adapt visually.

  • Year: 2009
  • Role(s): UI Designer and Developer
  • Agency / Client: Creative Corp

Challenges

Bringing in all the content

The site had to bring together on most pages multiple content types and allow each pod of information to work independently.

For example videos has a pod that changed every few seconds displaying relevant videos whilst the news pod would update with latest articles when they came in.

Design

Whilst the design was intentionally basic it offered up multiple issues when it came to getting the vision built within Drupal.

Technology

Jquery and Jquery UI were used to enhance and add interactive features to the site

Outcome

If the main aim was to be an information hub for all Temper Trap fans it served it purpose well. Whilst being quick to load it offered up information for eager fans quickly.
